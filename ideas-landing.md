# Ideas para la Landing Page — Justitia Law

---

## Secciones (facturables por separado)

Cada seccion es un bloque independiente que se cobra como unidad.

- [ ] **Sobre nosotros / El estudio** — Foto del abogado, biografia, experiencia, formacion, valores del estudio. Genera confianza y conexion personal.
- [ ] **Testimonios / Casos de exito** — Carrusel o grid con citas de clientes reales. Incluir foto (anonima si prefiere) y resultado del caso. Social proof critico para un estudio legal.
- [ ] **Proceso de trabajo** — Pasos visuales: 1. Contacto → 2. Consulta → 3. Evaluacion → 4. Solucion. Muestra al cliente que esperar y reduce la friccion.
- [ ] **Preguntas frecuentes (FAQ)** — Acordeon con preguntas tipo "¿Cuanto dura un divorcio?", "¿Que documentos necesito?". Util para SEO y para resolver dudas comunes sin que el cliente tenga que escribir.
- [ ] **Ubicacion / Mapa** — Embed de Google Maps con la direccion exacta en Huancayo. Importante para abogados locales.
- [ ] **Blog / Articulos** — Seccion con articulos legales breves (separacion de bienes, pension alimenticia, etc.). Excelente para SEO y para mostrar autoridad.
- [ ] **Equipo** — Si hay mas de un abogado, grid con fotos, especialidades y enlaces a LinkedIn o WhatsApp directo.
- [ ] **Precios / Paquetes** — Tarifas claras para consultas o servicios basicos. Transparencia que muchos clientes valoran.

---

## Mejoras / Ideas (sin complejidad, no son secciones nuevas)

Cambios pequeños que mejoran la experiencia sin requerir una seccion completa.

- [ ] **Meta tags & Open Graph** — Mejorar title, description, og:image, twitter:card para que se vea bien en enlaces compartidos (redes, WhatsApp). Bajo esfuerzo, alto impacto en presentacion.
- [ ] **Schema.org / JSON-LD** — Datos estructurados de tipo `LegalService` o `Attorney` para que Google muestre el estudio en busquedas con informacion directa (telefono, horario, reseñas).
- [ ] **Animaciones sutiles (scroll reveal)** — Los elementos aparecen suavemente al hacer scroll. Libreria liviana o CSS puro. Da sensacion premium.
- [ ] **Imagen lazy loading con placeholder** — Mientras carga la imagen principal mostrar un placeholder con color o blur.
- [ ] **Boton "volver arriba"** — Aparece al hacer scroll hacia abajo. Comodidad para el usuario.
- [ ] **Modo oscuro** — Alternador simple con CSS custom properties y `prefers-color-scheme`.
- [ ] **Boton de llamada directa** — Ademas del WhatsApp flotante, un boton `tel:+51940941857` para usuarios que prefieren llamar.
- [ ] **Notificacion de cookies / GDPR** — Banner simple si se usan analytics o se necesita cumplimiento legal.
- [ ] **Contador de visitas / estadisticas** — "Mas de 200 casos resueltos", "10+ anos de experiencia". Pequeno indicador numerico en el hero.
- [ ] **Favicon en formato multiple** — Agregar .ico, apple-touch-icon, etc. Detalle tecnico que suma profesionalismo.
- [ ] **Performance: preload de fuentes e imagenes clave** — Usar `<link rel="preload">` para que las fuentes y la imagen del hero carguen mas rapido.
- [ ] **Formulario con validacion** — Feedback visual si el usuario no llena un campo requerido antes de enviar.
